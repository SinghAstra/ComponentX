## Code Blocks with Syntax Highlighting (rehype-prism)

### JavaScript Example

```javascript
function demonstratePlugins(features) {
  return features.map((feature) => {
    console.log(`Exploring: ${feature}`);
    return `Enhanced ${feature}`;
  });
}

const pluginFeatures = ["markdown", "syntax", "transformation"];
demonstratePlugins(pluginFeatures);
```

### Python Example

```python
def markdown_magic(content):
    """Transform markdown with ease"""
```
